crayon-lang-vbnet
=================

Visual Basic .Net Syntax for Crayon Syntax Highligher

Installation
------------

1. Search for the folder "langs" inside your Crayon installation.
2. Create a folder named "vbnet".
3. copy **ALL** Text files into that folder.
4. Done.
